# Future Ideas

## Short-term roadmap

- Static PixiJS companion stage with a small, disposable asset lifecycle.
- Centralized settings persistence and accessibility preferences.
- Coverage reporting and one Playwright Electron journey.
- Branded tray and installer assets.

## Medium-term roadmap

- Deterministic animation state engine with reduced-motion support.
- Consent-driven Codex requests with context preview and action approval.
- Push-to-talk voice adapter and one developer integration.

## Long-term roadmap

- Signed, isolated plugin platform with publisher trust and capability quotas.
- Privacy-preserving local memory and optional encrypted synchronization.
- Production update channels, crash recovery, and multi-profile support.

## Nice-to-have features

- Multiple companion themes, workspace profiles, keyboard shortcuts, and per-monitor placement profiles.
- Custom tray menus, import/export of preferences, and richer accessibility narration.

## Stretch goals

- Declarative third-party companion packs.
- Cross-device continuity with explicit privacy controls.
- Optional collaboration surfaces for shared developer workflows.
